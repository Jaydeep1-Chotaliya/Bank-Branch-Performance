## 📚 Glossary of Financial Terms

| Term | Meaning |
|------|---------|
| **NPA** | Non-Performing Asset – a loan where payments are overdue by 90+ days |
| **LD Ratio** | Loan-to-Deposit Ratio – total loans ÷ total deposits × 100 |
| **FY** | Financial Year – the 12-month accounting period |
| **YoY** | Year-over-Year – comparing data to the same period last year |
| **MoM** | Month-over-Month – comparing data to the previous month |
| **Asset Quality** | A measure of loan portfolio health – low NPA = good quality |
| **Provisions** | Funds set aside to cover potential loan losses |
| **CASA** | Current Account and Savings Account – low-cost deposits for banks |
| **Credit Risk** | Risk of borrower defaulting on loan payments |
| **Liquidity** | A bank's ability to meet short-term obligations |
| **Net Profit Margin** | Strong profitability – significantly above industry average of 15-25% |
| **Yield on Loans** | Healthy lending returns within industry benchmark of 8-15% |
| **Cost-to-Income Ratio** | Efficient operations – below the ideal threshold of 60% |
| **ROA** | Indicates effective asset utilization |
| **ROE** | Reflects strong shareholder value creation |
| **CAGR** |  Demonstrates consistent growth trajectory |
